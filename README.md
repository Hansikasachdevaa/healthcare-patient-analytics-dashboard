
# Healthcare Patient Analytics Dashboard

## Overview
This project analyzes **55,500 patient records collected over 5 years** to understand hospital operations, patient demographics, and treatment trends.  
The analysis focuses on admissions, billing patterns, diagnostic outcomes, and disease prevalence using an interactive **Power BI dashboard**.

## Dataset Summary
- Total Records: **55,500 patients**
- Time Period: **5 Years**
- Doctors: **40,000**
- Hospital Rooms: **400**
- Average Patient Age: **51.54 years**
- Average Billing Amount: **~$26,000**
- Average Length of Stay (LOS): **15.51 days**

## Key Analysis Areas
- Patient demographics (age, gender, blood type)
- Hospital admissions trends
- Diagnostic test results (Normal, Abnormal, Inconclusive)
- Disease distribution
- Medication usage
- Hospital billing and revenue trends
- Impact of Length of Stay on hospital capacity

## Key Finding
Around **35% of diagnostic results were inconclusive**, indicating potential **delays in treatment and the need for improved early disease detection technologies**.
